Compilez le projet en double cliquant roguelike.jar present à la racine du projet.
une fois le jeu lancé, appuyé sur :
LEFT: pour se deplacer à gauche
RIGHT: pour se deplacer a droite
UP: pour monter en haut
DOWN: pour descendre
ESPACE: pour sauter sur un trou
n : pour eteindre le feu(sans les capsules d'eaux vous perdez en vie en vous approchant du feux)
- vous êtes parfois améner à prendre des boules casseurs(boule bleu) pour traverser certains mur(type cassable)
